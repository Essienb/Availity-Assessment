2. Tell me about something you have read recently that you would recommend and why. (Can be a Github Repo, Article, Blog, Book, etc):

I was studying about RabbitMQ. RabbitMQ is the most widely deployed open source message broker. One of the reasons is because it is lightweight and easy to deploy on premises and in cloud. It supports multiple messaging protocols examples like Messaging Queue Telemetry Transport(MQTT), Simple Text-Oriented Messaging Protocol(STOMP), and Advanced Message Queue Protocol(AMQP).

Some of the feature of RabbitMQ is that it supports Transactional messaging, messaging as a Cloud Service, allows you the publish and subscribe, automatic protocols assigment, real time events feed. The outstanding thing about it is the fact that you can send enormous volume of messages while still receiving status updates because of its asynchronous messaging capability. Through this study I was able to built two Applications that used RabbitMQ to deliver messages with different topics to various destinations.
